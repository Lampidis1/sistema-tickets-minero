# Sistema de Tickets Minero

Aplicación web para gestión de tickets de mantenimiento de equipos mineros.

## Características

- Dashboard en tiempo real
- Gestión de tickets
- Inventario de equipos
- Integración con WhatsApp
- Almacenamiento de fotos (5 años)

## Tecnologías

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS
- Supabase

## Deploy

